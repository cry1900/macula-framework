## macula-cloud

macula提供的与云端应用的连接器

### macula-cloud-base

连接Macula基础后台，包括权限数据、参数枚举等信息,实例基本信息上传等。

### macula-cloud-apm

连接应用性能平台

### macula-cloud-batch

连接批处理平台

### macula-cloud-msg

连接消息推送中心

### macula-cloud-nacos

连接配置中心

### macula-cloud-oss

连接存储中心

### macula-cloud-sentinel

连接服务治理平台

### macula-cloud-stream

连接消息流平台

### macula-cloud-tx

连接分布式服务平台
